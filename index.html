Lucier — how to use

Choose a source: the generated voice approximation, an uploaded audio file, or a microphone recording (up to 6 seconds). Press play to hear the original. Press + iterate to apply one pass of resonant room filtering. Each iteration reinforces the space's characteristic frequencies while attenuating everything else. The volume is normalised after each iteration so levels stay consistent across 10, 15, or more passes. Press reset to return to the original at any time.

The loop button makes the current buffer repeat continuously. Use this to sit with a single iteration before deciding whether to process it further.

The download section lets you export any range of iterations as a single WAV file, optionally looped. Select from iteration, to iteration, and number of loops. File size is shown as an estimate before rendering.

Radigue — how to use

Press play to start the drone. Three oscillators — tuned slightly apart from each other — produce slow beating at their difference frequencies. Each oscillator has its own LFO running at a different rate, so the frequency relationships drift continuously over time. The result is a sound that changes too slowly to notice moment to moment, but is clearly different after several minutes.

Base freq sets the fundamental pitch. Detuning controls how far apart the oscillators are — wider detuning produces faster, more audible beating; narrower produces slower, more subtle movement. Drift rate controls how quickly the LFOs modulate the frequencies. Noise floor adds a thin layer of pink noise beneath the tones, giving them a sense of physical presence.

The download renders a WAV file of the specified duration using the current settings. The LFO drift is modelled in the offline render, so the exported file captures the actual slow evolution rather than a static snapshot.

Try it in a real room

The physical version of the Lucier process requires only two devices. Record yourself speaking on one device. Play that recording back in a room through a speaker and record it on a second device. Play the second recording back and record it on the first device again. Repeat four or five times. Each pass through the room reinforces its resonant frequencies. Consistent speaker and microphone placement matters — the closer the microphone is to a wall or corner, the more strongly the room modes will be reinforced.
